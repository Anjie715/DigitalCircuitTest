User Manual
1. Open PyCharm or any other IDEs or with cmd to run the ProjSim.py.
2. Put one input vector into the input.txt file. Should be just one vector each time.
    Vector should look like this: 1110101
3. Run the program.
4. Enter the circuit file name.
    File name should be in the same format as this: 'filename.txt' (two apostrophe ' ' cannot be omitted!)
5. Enter the largest line(wire) number in this circuit. 
    For example, for 's27.txt' is 20, for 's298f_2.txt' is 202, 's344f_2.txt' is 190 and 's349f_2.txt' is 189.
6. The output vector will be in the output.txt.
